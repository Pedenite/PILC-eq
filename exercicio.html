a) 3*(34-8) + 24/4 = x
3*26 + 6 = x Respeitamos a ordem das operações
78 + 6 = x Continuamos a resolução seguindo a ordem
x = 84 Apenas invertemos o resultado final com x
b) 8 + x/3 = 34 + 8/4
x/3 = 26 + 2 Subtraímos ambos os lados por 8 para isolar x
x = 28*3 Continuamos a resolução seguindo a ordem dos operadores e multiplicamos os dois lados por 3
x = 84 Obtemos a resposta com o x isolado
c) 4x + 12 = 0
4x = -12
x = -12/4
x= -3
d) 5x - 6 = 2x + 3
5x - 2x = +9
x = 9/3
x = 3
e) (x+2)/3 = (2x+1)/5
5(x+2) = 3(2x+1)

5x - 6x = -(+3) - 10

Equações do 2° grau

Resolva as Questões Abaixo:

Dica: É possível abrir a questão para visualizar a solução

a) 5x^2 - 3x - 2 = 0
a= 5 b=-3 c=-2 Obtemos os valores de 'a', 'b' e 'c' a partir da equação
Δ = b^2 - 4ac Vamos aplicar os valores nesta fórmula para obter delta
Δ = (-3)^2 - 4*5*(-2) Ficaremos com esta expressão
Δ = 9 -20*(-2) Desenvolvemos as operações
Δ = 9 + 40 Desenvolvemos as operações
Δ = 49 Obtemos o resultado de delta
x1 = -b + √(Δ)/2a e x2 = -b - √(Δ)/2a Vamos aplicar os valores nesta fórmula para obter as raízes da equação
x' = -(-3) + √(49)/10 e x" = -(-3) - √(49)/10 Substituindo os valores na fórmula
x' = 3 + 7/10 e x" = 3 - 7/10 Resolvemos as operações em partes
x' = 1 e x" = - 4/10 Encontramos as raízes da equação
b) x^2 - 6x + 5 = 0
a= 1 b=-6 c=5
Δ = (-6)^2 - 4*1*(5)
Δ = +36 - 4 - 1*(5) ou Δ = +36 + 4 - 1*(5)
Δ = 36 - 20
Δ = 16
x' = -(-6) + √(16)/2 e x" = -(-6) -√(16)/2
x' = 6 + 4/2 e x" = 6 - 4/2
x' = 5 e x" = 1
